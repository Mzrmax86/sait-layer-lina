# Сайт «Ангелина Белова — судебный юрист»

Статический сайт: работает на любом хостинге, не требует баз данных и программистов.

## Как открыть сайт на компьютере

Откройте файл `index.html` двойным щелчком — сайт откроется в браузере.
(Для полной проверки формы и шрифтов нужен интернет.)

## Как редактировать содержимое — БЕЗ программиста

**Весь контент сайта находится в одном файле: `js/data.js`.**
Откройте его в любом текстовом редакторе (Блокнот, Notepad++, TextEdit) и меняйте текст внутри кавычек `"..."`. 
После сохранения обновите страницу в браузере.

Что где лежит в `data.js` (разделы пронумерованы прямо в файле):

| Раздел | Что меняется |
|---|---|
| 1. Контакты | телеграм-канал, чат, личный аккаунт, инстаграм, ютуб, телефон |
| 2. Телеграм-бот | токен и chat_id для приёма заявок (см. ниже) |
| 3. Первый экран | манифест и подзаголовок |
| 4. Практики | блоки услуг |
| 5. Обо мне | биография, цитата, факты |
| 6. Видео | названия сюжетов и ссылки на них |
| 7. Кейсы | судебный опыт |
| 8. Вопросы и ответы | FAQ |
| 9. Отзывы | отзывы клиентов |
| 10. Документы | список шаблонов для скачивания |

### Как добавить новое видео

1. Сделайте стоп-кадр сюжета (скриншот) и положите картинку в папку `assets/video/`, например `cover-11.png`.
2. В `js/data.js` найдите раздел `6. ВИДЕО` и скопируйте один блок:

```js
{
  title: "Название нового сюжета",
  link: "https://ссылка-на-видео",
  cover: "assets/video/cover-11.png"
},
```

3. Вставьте его в список (не забудьте запятую после `}`), сохраните файл.

### Как добавить новый кейс

В разделе `7. СУДЕБНЫЙ ОПЫТ` скопируйте блок `{ ... },` и заполните:
`area` — практика, `title` — заголовок, `task` — что было, `result` — чем закончилось,
`sum` — сумма спора или примечание (например, `"Долг — 4,2 млн ₽"`).

### Как добавить документ для скачивания

1. Положите файл (PDF или DOCX) в папку `docs/`.
2. В разделе `10. ОБРАЗЦЫ ДОКУМЕНТОВ` добавьте блок с названием и именем файла.

Сейчас в `docs/` лежат файлы-заглушки `.txt` — **замените их настоящими шаблонами**.

## ⚠️ Перед публикацией сайта обязательно

1. **Замените кейсы и отзывы-примеры на реальные** — они помечены словом «ПРИМЕР» в `data.js`. Публиковать вымышленные кейсы и отзывы нельзя (закон о рекламе, ст. 5).
2. **Впишите реальные ссылки** на телеграм-канал, чат и соцсети (раздел 1 в `data.js`). Пока стоят заглушки `ВАШ_КАНАЛ`, кнопки аккуратно ведут на форму записи.
3. **Уточните черновые названия видео** — они написаны по стоп-кадрам, замените на настоящие названия сюжетов и вставьте ссылки на эфиры.
4. **Добавьте образование и стаж** в раздел «Обо мне» (раздел 5, там есть подсказка).
5. Замените `example.ru` на ваш домен в: `index.html` (метатеги), `robots.txt`, `sitemap.xml`.

## Как настроить приём заявок в телеграм

Форма «Записаться на консультацию» умеет отправлять заявки вам в телеграм:

1. В телеграме откройте бота **@BotFather** → команда `/newbot` → придумайте имя.
   BotFather выдаст **токен** вида `1234567890:AAAbbb...` — скопируйте его.
2. Найдите своего нового бота в поиске и нажмите **Start** (это обязательно).
3. Откройте бота **@userinfobot** — он покажет ваш **id** (число).
4. В `js/data.js`, раздел `2. ОТПРАВКА ЗАЯВОК`, впишите:

```js
telegramBot: {
  botToken: "1234567890:AAAbbb...",
  chatId: "ваш id"
},
```

Пока бот не настроен, форма предлагает посетителю написать вам напрямую.

**Замечание о безопасности.** Токен бота виден в коде страницы — это особенность сайтов без сервера. 
Используйте этого бота **только** для приёма заявок и не добавляйте его в чаты: 
даже если токен кто-то узнает, он сможет лишь писать вам сообщения. 
Если потребуется полная защита — закажите у хостинга «серверную функцию» (Cloudflare Workers, Vercel) и перенесите токен туда.

## Как «доступ к документам после подписки» работает

Сайт без сервера не может проверить подписку технически, поэтому используется честный приём:
кнопка «Подписаться на канал» открывает телеграм → посетитель возвращается и нажимает
«Я подписался(ась)» → раздел скачивания открывается (и запоминается в его браузере).

## Аналитика посещаемости

1. **Яндекс.Метрика:** зарегистрируйте счётчик на [metrika.yandex.ru](https://metrika.yandex.ru), получите номер.
2. **Google Analytics:** создайте поток на [analytics.google.com](https://analytics.google.com), получите идентификатор `G-...`.
3. Откройте `index.html`, найдите блоки «АНАЛИТИКА ПОСЕЩАЕМОСТИ» в начале файла,
   уберите обёртку комментария `<!--` и `-->` вокруг нужного блока и вставьте свой номер вместо `XXXXXXXX`.

После этого в кабинетах Метрики и Analytics будет видно: сколько людей заходит, откуда они приходят (поиск, телеграм, соцсети), что смотрят.

## Как опубликовать сайт в интернете

Любой из вариантов:

- **Обычный хостинг** (reg.ru, timeweb и т.п.): загрузите все файлы папки в корень сайта по FTP или через панель.
- **Бесплатно — GitHub Pages / Netlify / Vercel:** перетащите папку сайта в интерфейс Netlify Drop — сайт сразу получит адрес; 
потом можно привязать свой домен.

После публикации добавьте сайт в [Яндекс.Вебмастер](https://webmaster.yandex.ru) и 
[Google Search Console](https://search.google.com/search-console) — это ускорит попадание в поиск. Sitemap уже подготовлен: `sitemap.xml`.

## Структура файлов

```
index.html        — страница сайта (контент подтягивается из data.js)
privacy.html      — политика конфиденциальности (нужна для формы)
css/style.css     — оформление
js/data.js        — ВЕСЬ КОНТЕНТ — редактировать здесь
js/main.js        — логика сайта (не трогать)
assets/photo/     — ваши фотографии
assets/video/     — обложки видео
assets/logo/      — логотип Первого канала
docs/             — файлы шаблонов для скачивания
robots.txt        — указания поисковикам
sitemap.xml       — карта сайта для поисковиков
```
